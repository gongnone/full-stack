# The Agentic Content Foundry

A full-stack content generation platform built on Cloudflare Workers, featuring AI-powered content pipelines, multi-client agency operations, and real-time collaboration.

## Tech Stack

- **Runtime**: Cloudflare Workers
- **Database**: Cloudflare D1 (SQLite)
- **Storage**: Cloudflare R2
- **Auth**: Better Auth with OAuth (Google, GitHub)
- **Frontend**: React + TanStack Router
- **API**: tRPC + Hono
- **ORM**: Drizzle
