## Core Definition

BMAD-METHOD is "an AI-driven development orchestration framework distributed as an NPM package (`bmad-method`) that provides structured, guided workflows powered by specialized AI agents."

Unlike generic coding assistants, it implements a complete agile methodology with intelligent routing, specialized agent teams, and battle-tested workflows.

## Key Characteristics

| Aspect | Details |
|--------|---------|
| **Package** | `bmad-method` on NPM registry |
| **Version** | v6.0.0-alpha.16 (major architecture revision) |
| **Distribution** | CLI via `npx`, IDE integrations, web bundles |
| **Core Foundation** | BMad Core (C.O.R.E. Framework) |
| **Primary Module** | BMad Method (BMM): 12 agents, 34 workflows, 4-phase lifecycle |

## Installation & Distribution

### Install Command
```bash
npx bmad-method@alpha install
```

### Directory Structure
Creates unified `_bmad/` directory with update-safe customization through `_config/` preservation system.

### Distribution Channels

1. **NPM** - CLI installation
2. **Web Bundles** - ChatGPT/Claude integration
3. **GitHub Releases** - Offline installation
